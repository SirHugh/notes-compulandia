# Gestión interna de Cotización y Compras. 
Este documento expone la forma como actualmente se gestionan las solicitudes de asesores para cotización de productos para la venta de productos a clientes.

#### Origen de las solicitudes de cotizacion y compra.
1.  Cuando un cliente manifiesta interés por un producto que no se encuentra en stock.
2.  Cuando se realiza una compra de un producto solo online, en alguna plataforma de ecommerce, ya sea la web propia, tienda naranja o contimarket. 

Proceso de negocio basico de cotizacion:
1.  cliente manifiesta interes por un producto(pregunta por presupuesto, precio, disponibilidad)
2.  vendedor comunica a compras el interes del cliente (tablero de notion)
3.  compras averigua sobre el producto responde sobre disponibilidad y precio. y responde
4.  vendedor comunica esto al cliente.
5.  el cliente confirma o rechaza. 
6.  si hacepta se pasa a un flujo de compra del item confirmado.
7.  si se rechaza se puede proponer otro producto, e iterar sobre los de este flujo.

# Flujo de recepción de equipo para reparación.
	El cliente presenta su equipo al tecnico recepcionista listando los problemas que tiene el equipo. 
	El Tecnipo determina el problema de acuerdo a lo que el cliente manifiesta.
## Alta del servicio.
-  Alta del cliente
-  Alta de la tarjeta del equipo. 
	-  Nombre 
	-  Numero de serie 
-  Alta de lo que el cliente manifiesta
-  Alta de accesorios que vienen con el equipo
-  Alta de la contraseña si se requiere. 

## Operativa
 -  Luego del alta del servicio no se asigna a un técnico. 
 -  Un técnico toma el trabajo y lo diagnostica
 -  El diagnostico se envía a un grupo de google chat al encargado. 
 -  El encargado cura el diagnostico y envía el presupuesto al cliente

### Problema
No se utiliza el camban para flujo de procesamiento de los servicios. 
Los servicios quedan en estado abierto y luego ya a cerrado. Los técnicos no pasan las columnas. 
Los servicios no se asignan a los técnicos.

#  Informe de Relevamiento en taller. 
#### Sobre la asignación del servicio. 
-  El servicio se asigna de forma operativa, no sistémica. El supervisor anota en una planilla excel los datos del servicio, a quien se asigno, unos parámetros de dificultad de la tarea y duración. 
-  El coordinador no tiene una lista de trabajos asignados por técnico. 
-  Manejan un concepto de prioridad de forma verbal. El supervisor asigna varios servicios a un técnico y una prioridad a las actividades.
-  El supervisor no puede ver todos los servicios que tienen asignado cada técnico. 

#### Sobre diagnostico, presupuesto y aprobación. 
-  Los técnicos utilizan un grupo de chat de google para enviar sus reportes de diagnostico, evaluación y sugerencia sobre el servicio. 
-  Los técnicos utilizan una misma maquina para la carga del diagnostico. 
-  El supervisor recibe el mensaje, toma y lo transforma en un presupuesto para el cliente y lo envía. 
-  El supervisor recibe la aprobación del cliente por whatsapp.
-  La notificación sobre la aprobación sobre el trabajo propuesto por el técnico se hace de forma verbal entre el superviso y el técnico. 

#### Sobre la culminación del servicio.
-  El técnico determina la finalización del servicio.
-  El técnico asignado avisa de forma verbal al supervisor sobre la terminación del servicio. 
-  El Supervisor verifica y es el encargado de cerrar el servicio. 
-  El Cajero o recepcionista es el encargado de crear el pedido para el cliente. 
-  Actualmente el pedido se crea al final del servicio, cuando el cliente viene para pagar y retirar.
